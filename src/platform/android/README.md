# Android Platform Path

Android is the primary target path for Phase 0.

Future work should add:

- Android native activity or app glue.
- CMake/Gradle integration using Android NDK.
- Vulkan instance creation with Android surface support.
- Device capability query on the Samsung Galaxy S26 Ultra.
- On-screen diagnostic overlay.
- Capability report written to an Android-accessible app location.
- Unsupported-device screen with extension/feature details.

Do not use browser, WebView, WebGPU, or fake RT for this project.
