# Capability Report Tools

This folder is reserved for future helper scripts or utilities that inspect, validate, copy, or summarise capability reports.

Phase 0 app output should eventually include JSON and/or text containing:

```text
Backend: Vulkan
RT mode: RayTracingPipeline / RayQuery / Unsupported
GPU name
Vendor ID
Device ID
Driver version
Vulkan API version
VK_KHR_acceleration_structure: yes/no
VK_KHR_ray_tracing_pipeline: yes/no
VK_KHR_ray_query: yes/no
VK_KHR_buffer_device_address: yes/no
VK_KHR_deferred_host_operations: yes/no
Internal render resolution
FPS / frame time
```

This tool folder should not fake capability values. It should only process reports generated by the real native Vulkan app.
