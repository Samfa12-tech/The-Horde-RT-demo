# Assets

The running alpha uses the Hotstrike Studio skeleton derivative, five CC0 Poly Haven material sets, thirteen FilmCow WAVs, and project-created branding/icon art. The skeleton was textured, rigged, and animated with Meshy. Separate Meshy sword and torch studies remain staged under `models/` until a static GLB/PBR path is measured on phone.

Future additions must stay commercial-safe, licensed, bounded, and measured on the Android RT path. Do not expand the shipped audio/model set simply because source assets are available.

Every asset must be recorded in `ASSET_LICENSES.md` before use.
