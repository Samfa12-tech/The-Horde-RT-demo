# Assets

The merged-animation skeleton and five CC0 Poly Haven material sets are loaded by the running demo. The Meshy PBR sword remains staged under `models/` until a static GLB/PBR path is measured on phone.

This project will eventually need commercial-safe historical gothic assets: wet stone, corridors, torches, lanterns, fog/smoke effects, ruins, goblins/gremlins, SFX, and possibly Pocket Chordsmith music.

Every asset must be recorded in `ASSET_LICENSES.md` before use.
